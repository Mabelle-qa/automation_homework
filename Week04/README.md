# Python 程式設計基礎：Chapter 7 實務練習

本專案收錄了 Python 程式設計課程第七章的兩份核心作業。內容涵蓋了基礎的條件判斷、迴圈處理、異常處理（Exception Handling），以及模擬實務開發中的 API 自動化測試邏輯。

## 📁 作業內容說明

### 1. 數字過濾器 (基礎題) - `homework01.py`
**題目目標：** 建立一個與使用者互動的程式，對輸入的數字進行篩選與分類。
- **功能邏輯：**
  - 判斷數字為 **偶數** 或 **奇數** 並印出對應符號。
  - 實作 **防呆機制**：確保輸入為正整數，若格式錯誤則顯示警告。
  - **迴圈控制**：若輸入數字超過 `50`，程式將使用 `break` 立即停止執行。
- **學習重點：** `while` 迴圈、`if-elif-else` 判斷式、`try-except` 格式檢查。

---

### 2. API 組合測試矩陣 (進階挑戰題) - `homework_en.py`
**題目目標：** 模擬企業級開發中「多使用者 × 多 API 端點」的自動化整合測試腳本。
- **功能邏輯：**
  - **模擬請求**：呼叫副程式隨機產生 `200 OK`、`404` 或 `500` 等 API 回傳狀態。
  - **動態重試 (Retry)**：根據使用者規模自動調整每個請求的最大重試次數（連動 `len(users)`）。
  - **早停機制 (Early Exit)**：任一 API 測試失敗，該使用者即標記為「❌ 未通過」並終止後續測試。
  - **結果報表**：自動分類並產出格式化的測試結果彙整報告。
- **學習重點：** 巢狀迴圈 (Nested Loops)、模組化開發 (Module Import)、狀態旗標 (Flag) 運用、複雜資料結構處理。
