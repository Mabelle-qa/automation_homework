📝 作業 ：下注自動決策 CLI 工具（實務版）
🎯 題目需求
請設計一支模擬下注決策的自動化腳本，需具備以下功能：

輸入：
年齡、餘額、是否驗證、是否為 VIP、是否封鎖

條件檢查：
年齡 >= 18
餘額 >= 100
已驗證
未被封鎖

策略輸出：
VIP 且餘額 >= 2000 → 🔥 超高風險策略
VIP 且餘額 >= 1000 → 💎 高風險策略
餘額 >= 1000 → 🥇 穩定策略
餘額 >= 500 → 🥈 中階策略
其他 → 🔰 保守策略

額外要求：
需使用函式模組化
需有防呆機制
需輸出決策報告（年齡 / 餘額 / 策略 / 是否可下注）
