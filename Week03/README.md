📘 Week 3 作業：下注自動決策 CLI 工具（實務版）

本專案為 Week 3 的作業內容，以 Python 實作一套簡易的「下注決策自動化 CLI」工具。
使用者透過終端機輸入資料後，系統會自動進行條件檢查並輸出最適合的策略建議。

🎯 作業內容說明

請設計一支「模擬下注決策」的自動化腳本，需具備以下功能：

🧩 1. 使用者輸入資料

依序輸入下列五項資訊（以 英文逗號 , 分隔）：

年齡（Age）

餘額（Balance）

是否驗證（y/n）

是否為 VIP（y/n）

是否封鎖（y/n）

🧪 2. 條件檢查項目

系統需驗證使用者是否符合下注資格：

年齡 ≥ 18

餘額 ≥ 100

帳號 已驗證

帳號 未被封鎖

若任一條件不符合，系統會即時提示錯誤訊息並停止後續流程。

🧠 3. 策略輸出規則

符合基本資格後，依照使用者身分（VIP / 非 VIP）與餘額輸出對應策略。

🧑‍💼 若使用者為 VIP：
餘額（Balance）	策略分類
≥ 2000	🔥 超高風險策略
≥ 1000	💎 高風險策略
👤 若使用者非 VIP：
餘額（Balance）	策略分類
≥ 1000	🥇 穩定策略
≥ 500	🥈 中階策略
其他	📘 保守策略
📤 4. 系統輸出報告（必須包含）

年齡（Age）

餘額（Balance）

策略（Strategy）

是否可下注（需通過條件檢查）

📌 額外要求

使用 函式模組化（Modularization）

必須具備 防呆機制（輸入錯誤需提示）

以 CLI互動方式 完成整體流程設計

▶️ 使用方式
步驟 1：執行程式
python homework.py

步驟 2：依提示輸入資料

例如：

18, 1200, y, n, n


系統即可依照規則輸出對應策略。
